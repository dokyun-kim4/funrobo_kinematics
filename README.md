## Project Overview
In this project, we derived both analytical and numerical solutions for inverse kinematics for two robot manipulator platforms: HiWonder's 5DOF arm and Kinova's 6DOF arm. The analytical and numerical implementations were tested in simulation for both platforms. For the HiWonder arm, they were also tested on real hardware.
